# Cramer-s-Method-of-Linear-Algebra
A program that solves linear equation systems (in 2x2 and 3x3 equations) using the cramer method

**Let's solve the following system of equations using Cramer's rule.**

*2x2 equation* <br/> <hr/>
4x - y = 5 <br/>
x + 2y = 8

*3x3 equation* <br/> <hr/>
3x + 2y - z = 9 <br/>
y - z = 4 <br/>
2x + y + z = 2 <br/>
